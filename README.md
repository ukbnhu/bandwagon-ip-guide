这是被问最多的问题，答案是：**可以，但有一个很关键的限制。**

搬瓦工的机房迁移功能（Migrate to another DC）是免费的，次数不限，迁移后 IP 地址会随之改变。理论上这等于"免费换 IP"。

你试着操作，页面会显示错误，但不会告诉你是因为 IP 被封了。很多人在这里搞不清楚状况，来回点来点去，最后发现就是死活迁不了。

如果 IP 还没被封，提前迁机房是完全可以的，也没有成本。

### 迁机房步骤（简版）

1. 登录 KiwiVM 控制面板
2. 在主页面停止 VPS（stop）
3. 左侧菜单找到 "Migrate to another DC"
4. 选择目标机房，确认迁移
5. 等待几分钟到十几分钟，迁完会发邮件通知

整个过程数据保留，不会丢失。迁完之后记得更新你的 SSH 配置里的 IP 地址。
